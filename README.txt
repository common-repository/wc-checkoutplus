The complete WooCommerce checkout field editor. Add, edit, re-order, validate and conditionally display fields.

== Description ==
A WooCommerce checkout field editor that is very easy to use. You can add unlimited custom fields to your WooCommerce checkout page. You can manage, enable, disable and customize the features of default fields as well.

### Why this WooCommerce Checkout Field Editor?

#### Unlimited custom fields

No restriction on the number of custom fields you can add to your checkout page. Our plugin is free and you have access to add unlimited number of custom fields to Billing, Shipping and Additional fields sections.

#### Access to all types of fields

Unlimited access to **ALL** field types from text input to HTML fields, including checkboxes and radios. The following filed types are currently available and more are being added regularly

~~~~
* Text
* Textarea
* Select
* Radio
* Checkbox
* Password
* Email
* Date
* HTML
* Hidden Input
~~~~

#### Re-arrange woocommerce checkout fields

Determine the order in which your checkout fields will show up. You can simply drag the fields up and down to your desired location.

#### Sleek Admin Interface

We use sleek admin interface built in VueJS using cutting-edge technology to make your experience beautiful. It's responsiive and displays well on all devices.

#### In-built Documentation Links

Right from your admin dashboard, you can conveniently browse documentation pages and click to learn more.

#### Quick & Fast Support

This plugin is free but we reply support tickets very quickly and politely. Do you have any question or want to know more, please click on the Support button above.

#### Conditional Fields [^1]

You can choose to make fields appear if other fields are available or have certain values. You can add multiple and unlimited custom fields to Checkoutplus woocommerce custom field editor.

#### 45+ Validation Rules [^1]
By default WooCommerce comes with a few validation rules like `required` and `phone` but here we offer over 45+ custom validation rules.

~~~~
If you only want to accept numbers you can just use `numerical` rule.
If you want to accept only letters, you can use `alpha` rule
If you want to accept date in any format, you can use the `date` rule.
If you want regular expression (regex), you can use it too.
~~~~

#### Page Display [^1]
You can choose to display your checkout fields values in your emails or order details page. Lots of options and possibilities with Checkoutplus - WooCommerce checkout field editor.

For more information, please visit **validation** section here - [Documentation](https://pluginette.com/knowledgebase/checkoutplus/)

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wc-checkoutplus` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to WooCommerce/CheckoutPlus Fields

== Frequently Asked Questions ==

= How do I access this editor? =

Once installed please go to **WooCommerce > Checkout Fields** and you will see your CheckoutPlus editor

= Are all the features above free? =

All features are free except there is a footnote on it saying it's a pro feature.
